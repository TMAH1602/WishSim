# 2026-07-20 — v0.5.0 feature release

- Publish the game shell, persistent teams, Character Management, weapon equipment, three-star raster weapon art, Corvin complexion correction, and final ten-pull skip behavior documented in `2026-07-20-game-shell-teams-equipment.md`.
- Preserve Ghostty as the primary supported terminal while retaining Kitty protocol and ANSI fallback support.
